# MakeMyTrip Automation Framework

Project Overview

This project automates flight search functionality on MakeMyTrip using Selenium + Java + TestNG.

##  Test Scenarios

* Search flights from Delhi to Mumbai
* Validate search results page
* Verify filters functionality

##  Framework Design

* Page Object Model (POM)
* ThreadLocal WebDriver
* Extent Reports
* Screenshot on failure

## How to Run

1. Clone repo
2. Run testng.xml
3. Check reports in /reports folder
